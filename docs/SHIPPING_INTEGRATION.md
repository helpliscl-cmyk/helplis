# Shipping integration

Implemented now:

- `ShippingProvider` interface
- `ManualShippingProvider`
- manual carrier, service, tracking, cost and status registration in order detail

No carrier API is connected and no shipping rates are invented.

Future options should compare Chilexpress, Starken, Blue Express, CorreosChile and aggregator APIs by coverage, pickup workflow, tracking webhooks, labels, SLA and cost.
