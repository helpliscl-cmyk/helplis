# Competitor Benchmark

No se ejecutó scraping agresivo. Este documento deja una metodología legal para análisis público posterior.

Reglas:

- No copiar código, textos, logos, fotos ni identidad visual.
- No evadir autenticación ni acceder a paneles privados.
- No obtener datos personales ni vulnerar endpoints.
- Analizar solo páginas públicas, flujos visibles, precios públicos, experiencia móvil y comportamiento legítimo de navegador.
