# Organization Model

El modelo soporta colegios, jardines, academias, clubes, centros médicos, fundaciones, veterinarias, municipalidades, comunidades, empresas y otros.

Cada organización tiene slug, contacto, descuento, comisión, landing provisional, campañas, membresías y dispositivos asociados.

Implementado:

- `/admin/organizations`
- `/admin/campaigns`
- `/o/[organizationSlug]`

Pendiente: permisos finos por organización, landing con compra, reportes por convenio y CRM real.
