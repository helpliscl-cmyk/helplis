# Notification integration

Current provider remains local/mock and records `NotificationEvent`.

Prepared templates:

- request received
- order created
- payment pending
- payment reported
- payment confirmed
- order packing
- order shipped
- order delivered
- activation reminder
- activation completed
- device scanned
- location shared
- ticket received

Future providers:

- Resend
- SMTP
- WhatsApp Business API
- SMS
- push

No credentials are committed.
