# Operations runbook

Daily:

1. Check `/admin/operations`.
2. Review new leads and convert ready leads to orders.
3. Review payment reported orders.
4. Reserve available verified inventory for paid orders.
5. Complete packing and print activation cards.
6. Register manual shipment and tracking.
7. Review support tickets.

Production batch:

1. Create DEMO/SAMPLE batch.
2. Generate codes.
3. Export supplier package.
4. Import returned UID.
5. Verify units.
6. Move verified units to available inventory.

Emergency:

1. Export inventory CSV.
2. Export verification report.
3. Review audit logs.
4. Preserve generated package checksums.
