# Emergency Information

Fecha: 2026-07-13

HelPlis no es un servicio medico, no reemplaza servicios oficiales y no promete respuesta inmediata.

La accion "Informar emergencia" registra un evento secundario y muestra un aviso al scanner. Si hay riesgo inmediato, la UI indica contactar servicios oficiales.

La informacion critica autorizada puede incluir alergias, condiciones, medicamentos, instrucciones medicas, comunicacion, movilidad y sensorial. Cada campo se muestra solo si el responsable lo habilito.
