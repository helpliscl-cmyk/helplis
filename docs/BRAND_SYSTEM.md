# HelPlis Brand System

Fecha: 2026-07-11.

| Token | Valor | Uso |
| --- | --- | --- |
| `--brand-primary-dark` | `#0b3774` | Texto fuerte, botones principales, footer. |
| `--brand-primary-light` | `#00c1b6` | CTA, foco accesible, acentos de ayuda. |

## Tipografía

La base web actual usa Arial/Helvetica para continuidad con el MVP. Recomendación futura: evaluar Inter o Geist Sans si se quiere una apariencia más técnica sin cambiar el logotipo oficial.

## Componentes

- Botón primario: azul profundo, texto blanco.
- Botón accent: turquesa, usado para activación y compartir ubicación.
- Botón secundario: blanco, borde frío, texto azul.
- Formularios: altura mínima 44 px, foco turquesa, labels claros.
- Cards: radio 8 px, borde frío, sombra mínima. No anidar cards.
- Ficha pública: botones de acción grandes, marca secundaria, prioridad al nombre e instrucciones.

## Jerarquía

1. Estado o acción urgente.
2. Nombre de persona, mascota u objeto.
3. Instrucciones autorizadas.
4. Contacto y ubicación voluntaria.
5. Marca y contexto.

## Correcto

- Explicar QR + NFC con precisión.
- Decir explícitamente que no hay GPS propio ni rastreo constante.
- Usar la foto oficial de la pulsera para la home y OG.
- Mantener lenguaje simple, humano y chileno neutro.

## Incorrecto

- Usar el logo como patrón decorativo.
- Prometer seguridad garantizada, ubicación en tiempo real o protección total.
- Publicar precios, alianzas, testimonios o estadísticas no establecidos.
- Simular un producto GPS o vigilancia permanente.
