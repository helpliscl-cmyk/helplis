# GitHub Setup

Incluido:

- `.gitignore`
- `CONTRIBUTING.md`
- `SECURITY.md`
- `CODE_OF_CONDUCT.md`
- templates de issues y PR
- workflow `.github/workflows/ci.yml`

Pasos:

1. Crear repositorio remoto.
2. Agregar origin.
3. Push de rama principal.
4. Configurar protección de rama y required checks.
5. Agregar secretos solo en Vercel/Supabase, no en GitHub salvo que CI los necesite.
