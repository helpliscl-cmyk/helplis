# Contribuir a HelPlis

Este repositorio contiene un MVP local. Antes de proponer cambios:

1. Ejecuta `npm run lint`, `npm run typecheck`, `npm run test` y `npm run build`.
2. No agregues secretos, credenciales reales ni datos personales reales.
3. Mantén activation codes, UID NFC y password hashes fuera de UI, logs y exports públicos.
4. Documenta cualquier cambio que afecte privacidad, activación, importación o ficha pública.
